Returns Policy

Backdoor reviews return requests for eligible unworn items. Product condition, category, and order status all matter when a return request is submitted.

Eligible Returns

  • Items must be unworn, unaltered, and returned in the condition they were delivered.
  • Original box, accessories, tags, and packaging should be included when applicable.
  • Return approval is reviewed before a refund or store-credit outcome is issued.

Non-Returnable Cases

  • Used items, damaged items, or products missing original packaging can be declined.
  • Approved offers, special-order items, and some backorder pairs may be final sale.
  • Fraud screening or chargeback disputes can pause return handling until resolved.

How To Request A Return

Track the order and confirm delivery status Open your account dashboard and locate the order Contact support with the order number and reason for return
